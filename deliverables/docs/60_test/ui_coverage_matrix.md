# UI カバレッジマトリクス

## この文書の役割

| 項目 | 内容 |
|------|------|
| 目的 | 全 13 画面の UI コントロール x ロール x viewport の表示仕様を棚卸しし、既存テスト/smoke でカバーされていないギャップを特定する |
| 正本情報 | UI コントロール棚卸し、ロール別表示マトリクス、レスポンシブ挙動表、既存カバレッジマーキング、ギャップリスト |
| 扱わない内容 | バックエンド側の認可テスト、E2E テスト、API レベルのテスト |
| 主な参照元 | `50_detail_design/screens/*.md`, `50_detail_design/ui-guidelines.md`, `50_detail_design/authz.md`, `60_test/manual_checklists/smoke_check.md`, `expense-saas/frontend/src/pages/**/__tests__/*.test.tsx` |

## 凡例

| マーク | 意味 |
|--------|------|
| CT | component test（*.test.tsx）でカバー済み |
| SMK | smoke_check.md でカバー済み |
| CT+SMK | 両方でカバー済み |
| -- | 未カバー |
